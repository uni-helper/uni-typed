## 介绍

这个包是 `@uni-helper/uni-typed` 项目的一部分，将 uni-cloud 组件包装为可导入的 Vue v3 组件，并提供对应的 TypeScript 实例类型，请 [👉 查看 Netlify 文档](https://uni-typed.netlify.app/) 或 [👉 查看 Cloudflare 文档](https://uni-typed.pages.dev/) 了解具体用法和示例。
