## 介绍

这个包是 `@uni-helper/uni-typed` 项目的一部分，将 uni-app、uni-cloud 和 uni-ui 组件包装为可导入的 Vue v3 组件，并提供对应的 TypeScript 实例类型，即 [`@uni-helper/uni-app-components`](../uni-app-components)、[`@uni-helper/uni-cloud-components`](../uni-cloud-components) 和 [`@uni-helper/uni-ui-components`](../uni-ui-components) 三者的集合。请 [👉 查看文档](https://uni-typed.netlify.app/) 了解具体用法和示例。
