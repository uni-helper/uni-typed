# 类型同步记录（2026-06）

本次依据官方文档对类型定义进行了同步，核对范围与结论如下。所有差异均以官方文档原始 HTML 为准验证。

## uni-app-types

| 组件 | 改动 |
| --- | --- |
| `video` | 新增 `is-live`（`isLive`）属性 |
| `swiper` | 修正 `changing-class`（`changingClass`）类型为 `string`（原误写为 `boolean`） |
| `checkbox` / `radio` | 新增 `background-color`、`border-color`、`active-background-color`、`active-border-color`、`icon-color` 五个颜色属性 |
| `textarea` | 新增 `cursor-color`（`cursorColor`）、`inputmode` 属性及其枚举类型 |
| `switch` | 修正 `type` 枚举值：`_Switch` → `switch`（原值非合法属性值） |
| `picker-view` | 新增 `mask-top-style`、`mask-bottom-style` 属性 |
| `web-view` | 新增 `fullscreen` 属性，`@load`（`onLoad`）、`@error`（`onError`）事件 |
| `navigator` | 新增 `render-link`（`renderLink`）属性 |
| `icon` | 新增 `view-box`（`viewBox`）属性 |
| `scroll-view` | 修正 `show-scrollbar`（`showScrollbar`）默认值注释为 `true` |

`movable-view` 的 `@htouchmove`/`@vtouchmove` 属于平台相关的通用触摸事件，与 `view` 组件保持一致，未纳入类型（项目既有约定）。

## uni-ui-types

对全部 47 个 uni-ui 组件逐一核对，其中 13 个存在差异并已修正：

| 组件 | 改动 |
| --- | --- |
| `uni-number-box` | 新增 `width` 属性 |
| `uni-popup-dialog` | 新增 `maxlength`、`showClose`、`borderRadius` 属性 |
| `uni-search-bar` | 新增 `textColor` 属性 |
| `uni-segmented-control` | 新增 `inActiveColor` 属性 |
| `uni-transition` | 新增 `styles` 属性（复用既有 `_UniTransitionType`） |
| `uni-tooltip` | 新增 `placement` 属性及枚举 |
| `uni-nav-bar` | 新增 `showMenuButtonWidth` 属性，`@clickLeft`（`onClickLeft`）、`@clickRight`（`onClickRight`）事件 |
| `uni-file-picker` | 新增 `dir` 属性 |
| `uni-data-select` | 新增 `pageSize` 属性 |
| `uni-datetime-picker` | 新增 `@show`（`onShow`）事件 |
| `uni-easyinput` | 新增 `@keyboardheightchange`（`onKeyboardheightchange`）事件 |
| `uni-countdown` | 新增 `showHour`、`showMinute` 属性 |
| `uni-icons` | 新增 `@click`（`onClick`）事件 |

`uni-forms-item` 的 `label-position`/`validateTrigger`/`left-icon`/`icon-color` 在官方文档中标注「1.4.0 已废弃」，按项目约定不新增废弃 API。
